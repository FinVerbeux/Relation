# Vues d'enquête

Relation propose maintenant plusieurs vues complémentaires au graphe.

## Timeline

La timeline liste les événements du dossier du plus récent au plus ancien :

- création d'éléments ;
- modification d'éléments ;
- création de liens.

Chaque entrée peut être cliquée pour revenir directement sur l'élément ou le lien concerné.

## Vue tableau

La vue tableau permet de parcourir tous les éléments sous forme tabulaire avec :

- recherche locale ;
- filtre par type ;
- filtre par statut ;
- comptage des liens ;
- export CSV des éléments.

## Audit qualité

L'audit qualité détecte les points faibles d'un dossier :

- éléments sans source ;
- éléments confirmés sans source ;
- éléments isolés ;
- valeurs manquantes ;
- preuves fichier sans empreinte ;
- liens orphelins ;
- doublons possibles.

L'objectif est d'aider à transformer un graphe visuel en dossier exploitable et partageable.

## Exports CSV

Deux exports simples sont disponibles depuis l'interface :

- `*-elements.csv` pour les nœuds ;
- `*-liens.csv` pour les relations.

L'audit peut aussi être exporté séparément en CSV depuis sa modale.
